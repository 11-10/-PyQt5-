# -基于PyQt5的图片标注-
使用PyQt5对图片进行标注

修改config.py 文件
original_path = '20240708/data'  # 原始图片文件夹路径
results_path = '20240708/result'  # 标注结果存放文件夹路径
backup_path = '20240708/backup'  # 备份文件夹路径
运行main.py即可。
基于https://gitee.com/wb200327/image_tagger代码修改
